# Competitive Moats Quick Reference

## The 7 Powers Summary Table

| Power Type | Benefit | Barrier | Key Indicator | Duration |
|------------|---------|---------|---------------|----------|
| **Scale Economies** | Lower unit costs | Prohibitive cost of share gains | Cost advantage grows with scale | Permanent if maintained |
| **Network Economies** | Higher value from users | Insurmountable value deficit | Users attract more users | Very durable, but bounded |
| **Counter-Positioning** | Superior model economics | Incumbent cannibalization fear | Disrupting established model | Temporary (5-15 years) |
| **Switching Costs** | Customer captivity | High expected switching losses | Low churn, pricing power | Durable if maintained |
| **Branding** | Price premium | Decades of consistent association | Premium pricing accepted | Very durable if maintained |
| **Cornered Resource** | Enhanced value creation | Rival acquisition difficulty | Unique assets/talent/IP | Depends on resource |
| **Process Power** | Lower costs/better products | Long replication time required | Operational excellence | Very durable |

## Power Type Decision Tree

```
START: Does the company have any competitive advantage?
│
├─ NO → No Power present → Focus on operational efficiency only
│
└─ YES → What type?
    │
    ├─ Does unit cost decrease materially with scale?
    │   └─ YES → SCALE ECONOMIES
    │
    ├─ Does value increase as users increase?
    │   └─ YES → NETWORK ECONOMIES  
    │
    ├─ Is there a new model incumbents can't copy?
    │   └─ YES → COUNTER-POSITIONING
    │
    ├─ Would customers lose value by switching?
    │   └─ YES → SWITCHING COSTS
    │
    ├─ Can company charge premium for same product?
    │   └─ YES → BRANDING
    │
    ├─ Does company have unique resource access?
    │   └─ YES → CORNERED RESOURCE
    │
    └─ Is there an organizational advantage from time?
        └─ YES → PROCESS POWER
```

## Greenwald's Three-Part Test for Barriers to Entry

### Test 1: Market Share Stability
- Look at 5-10 year market share trends
- If shares stable despite competition → barriers likely exist
- If shares constantly shifting → no barriers

### Test 2: Sustained Excess Returns
- Calculate return on invested capital (ROIC)
- Compare to weighted average cost of capital (WACC)
- If ROIC > WACC for 5+ years → barriers likely exist

### Test 3: Incumbent vs Entrant Performance
- Do new entrants struggle while incumbents prosper?
- Are there high-profile failed entry attempts?
- Do incumbents maintain position despite poor management?

## Local vs Global Competitive Advantage

**Key Insight**: Competitive advantages are almost always LOCAL

### Geographic Locality
- Wal-Mart: Dominated regions before national expansion
- Nebraska Furniture Mart: Dominates Omaha area

### Product Space Locality  
- Microsoft: Started with PC operating systems only
- Intel: Dominates x86 microprocessors (exited memory chips)

### Expansion Strategy That Works
1. Establish local dominance
2. Consolidate position
3. Expand incrementally at edges
4. Repeat

### Warning: Global Ambition Without Local Dominance
- Apple (early): Tried to be both hardware + software globally
- vs Microsoft: Focused narrowly on OS, then expanded

## Power Intensity Formula

```
Power Intensity = [Industry Economics] × [Competitive Position]
```

### Industry Economics Factor
- How strong is the structural force creating the power?
- Scale: How much do costs decline with volume?
- Network: How much does value increase with users?
- Switching: How painful is switching?

### Competitive Position Factor
- How differentiated is the company's position?
- Scale: What is the scale advantage ratio?
- Network: What is the user base differential?
- Share: What is the market share gap?

## Quick Assessment Questions

1. **What would happen if a well-funded competitor entered?**
   - Easy to compete → No moat
   - Difficult/costly to compete → Potential moat

2. **Can the company raise prices without losing customers?**
   - Yes → Power present
   - No → No power

3. **Is the company's advantage getting stronger or weaker?**
   - Stronger with scale/time → Good power
   - Eroding → Temporary power

4. **Does the advantage transfer to adjacent markets?**
   - Yes → Expandable power
   - No → Limited power

## Moat Erosion Warning Signs

- Gross margins declining over time
- Market share slipping to new entrants
- Pricing power weakening
- Technology becoming commoditized
- Regulatory advantages expiring
- Key talent departing
- Network effects fragmenting
